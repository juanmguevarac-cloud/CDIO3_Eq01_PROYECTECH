# Archivos Mecánicos

Archivos Mecánicos – Chaleco Inteligente

Esta carpeta contiene la documentación relacionada con el diseño estructural y físico del sistema, incluyendo la carcasa, soportes de sensores y elementos de integración al chaleco.

El diseño mecánico se desarrolló considerando:

Ergonomía para el atleta

Distribución equilibrada del peso

Protección de componentes electrónicos

Facilidad de montaje y mantenimiento

📂 Contenido

Modelos STL – Archivos listos para impresión 3D.

Archivos DXF – Cortes y geometrías planas.

Diseños CAD – Archivos editables (SolidWorks, Fusion 360, etc.).

Planos técnicos – Dimensiones y tolerancias.

Versiones iterativas – Historial de mejoras y ajustes estructurales.



Integración segura con el textil del chaleco.

Optimización para prototipado rápido.
