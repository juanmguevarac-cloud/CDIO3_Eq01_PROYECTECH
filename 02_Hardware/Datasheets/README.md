# Datasheets
📑 Datasheets – Componentes Electrónicos

Esta carpeta almacena exclusivamente las hojas de datos oficiales (datasheets) de los componentes electrónicos utilizados en el proyecto.

Estos documentos son fundamentales para:

Verificar especificaciones eléctricas

Analizar consumo energético

Consultar diagramas de conexión

Validar rangos de operación

Diseñar correctamente el sistema de alimentación

📂 Componentes Documentados

ESP32-WROOM-32E – Microcontrolador principal

MAX30102 – Sensor de frecuencia cardiaca y SpO₂

BNO055 – IMU de 9 ejes

BMP280 – Sensor de presión y temperatura

GPS NEO-6M – Módulo de geolocalización

SIM800L – Módulo de comunicación GSM

TP4056 – Módulo de carga Li-ion

MT3608 – Convertidor Step-Up
